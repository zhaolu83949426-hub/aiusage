# AIUsage 扩展功能 PRD

**日期：** 2026-05-22
**版本：** v1.0

---

## F1：Hermes Agent 消耗支持

Hermes 是一款 AI 编程 Agent，当前 aiusage 尚无对应 parser，导致其 token 消耗无法被追踪。

- 新增 Hermes parser，扫描 Hermes 日志文件并解析每行 token 消耗
- 在 `Tool` 类型中添加 `'hermes'`
- 在 `discoverLogFiles` 中添加 Hermes 默认日志目录扫描
- 在 `SourcesConfig` 中添加 `'hermes'` 自定义路径支持

---

## F2：国产模型人民币计费 + 实时汇率

国产模型（DeepSeek、Qwen、GLM、Kimi 等）官方定价以 CNY 计价，当前价格表统一手动转换为 USD 导致汇率固化偏差；非国产模型继续使用 USD 计价。

- 价格表支持标注模型计价货币（`currency: 'CNY' | 'USD'`），国产模型标注 CNY，其余默认 USD
- 支持手动配置汇率，或从公开 API 自动获取并缓存
- 费用计算时按当前汇率将 CNY 转换为 USD 后再存入数据库，数据库始终存 USD
- UI 层支持切换显示货币（USD / CNY）

---

## F3：Skill / MCP 工具使用分析

`tool_calls` 表已记录所有工具调用，但未区分内置工具、MCP 工具和 Skills。

- 按命名规则分类工具类型（`mcp__` 前缀 → MCP，`Skill` → skill，其余 → 内置）
- Tool Calls 页面支持按类型过滤，MCP 工具展示 `server/action` 格式
- Overview 增加 MCP 服务器调用排名

---

## F4：会话详情（时长、阶段耗时、工具调用）

Sessions 页面当前只展示聚合统计，缺少单次会话内的时序细节。

- Sessions 列表新增时长列和工具调用数列
- 新增会话详情页，以时间轴展示 API 调用序列 + 各调用下的工具调用
- 标注相邻 API 调用之间的间隔时间

---

## F5：排行榜

### F5-A：个人时间轴排名（本地）

对单机用户即时可用，不依赖多用户同步。

- 按天/周聚合历史用量，生成个人历史排名
- 展示"今日用量是历史第 X 名"、个人最高记录、连续活跃天数等
- 类 GitHub 贡献热力图 + 个人记录系统

### F5-D：社区公开排行榜（可选，opt-in）

用户主动选择参与，匿名上报聚合统计数据至中央服务。

- 仅上报聚合数字（总 token、总费用、工具分布），不含任何会话内容
- 默认关闭，用户在 Settings 主动开启并设置展示名
- 服务端：Cloudflare Workers + D1，托管公开排行榜页面
- 本地客户端定期（每日）上报，展示全局排名

---

## F6：官方订阅配额 Dashboard

Claude Code Pro、Codex 等 AI 编程工具均有官方订阅配额限制（如 Claude Code Pro 的 5 小时滑动窗口），用户无法直观感知当前窗口内的剩余额度。

- 内置各工具官方订阅套餐的配额规则（窗口类型、token 上限或费用上限）
- 支持多账号：每个工具可配置多个账号，各账号独立计算配额消耗
- 新增 Quotas 页面，以进度条展示各工具 / 账号当前窗口的配额消耗，状态分绿 / 黄 / 红
- 主页可选展示配额超 80% 的警告提示
